# 智慧记AI进销存 - 用户使用指南

## 应用简介

**智慧记AI进销存** 是一款专为小微企业、个体商户设计的免费进销存管理软件。无会员限制、无广告、完全离线可用，所有数据存储在本地设备，保护您的商业隐私。

### 核心特点

- ✅ **完全免费** - 无会员限制，无广告打扰
- ✅ **离线可用** - 无需联网，数据本地存储
- ✅ **多单位支持** - 支持无限层级单位换算（如：箱→件→盒→个）
- ✅ **客户分级** - 自动客户等级管理，不同等级不同价格
- ✅ **智能分析** - 多维度统计分析，经营数据一目了然
- ✅ **蓝牙打印** - 支持蓝牙小票打印机
- ✅ **操作日志** - 所有关键操作可追溯

## 功能模块

### 1. 首页

首页提供快速入口和核心数据概览：

- **快捷入口**：销售开单、商品管理、客户管理、库存查询、销售记录、蓝牙设置
- **今日统计**：今日销售额、订单数、利润
- **待办提醒**：库存预警、应收账款、应付账款
- **最近销售**：快速查看最近5笔交易

### 2. 商品管理

#### 2.1 商品列表

- 支持按分类筛选
- 支持条码/名称搜索
- 显示库存和价格信息
- 点击商品可编辑

#### 2.2 商品多单位

支持无限层级单位换算，例如：

```
抽纸：包(基础) → 提(1提=3包) → 箱(1箱=30包)
矿泉水：瓶(基础) → 件(1件=24瓶)
```

每个单位可设置：
- 进价
- 零售价
- 批发价
- VIP价
- 至尊VIP价
- 库存预警值

#### 2.3 添加商品

1. 点击右上角"+"按钮
2. 填写商品基本信息
3. 添加至少一个基础单位
4. 可添加多个上级单位
5. 保存

### 3. 销售开单

#### 3.1 开单流程

1. **选择客户**（可选）
   - 点击客户头像选择
   - 系统自动应用客户等级价格

2. **添加商品**
   - 扫描条码（支持蓝牙扫码枪）
   - 或手动搜索添加
   - 选择单位和数量

3. **修改价格**（可选）
   - 点击商品行可改价
   - 系统记录改价日志

4. **结算**
   - 选择支付方式（现金/微信/支付宝/赊账）
   - 支持混合支付
   - 确认收款

5. **打印小票**（可选）
   - 连接蓝牙打印机
   - 自动打印小票

#### 3.2 销售记录

- 查看所有销售订单
- 支持按日期/状态筛选
- 点击查看详情
- 支持退货退款操作

### 4. 库存管理

#### 4.1 库存查询

- 实时库存数量
- 库存预警提醒
- 库存成本核算

#### 4.2 库存流水

- 详细记录每笔出入库
- 支持按类型筛选（销售/采购/退货/盘点）
- 显示操作时间和操作员

#### 4.3 库存盘点

- 实地盘点功能
- 自动计算盘盈盘亏
- 生成盘点报告

### 5. 客户管理

#### 5.1 客户分级

系统预设三个客户等级：

| 等级 | 折扣 | 升级条件 |
|------|------|----------|
| 普通客户 | 无折扣 | 默认 |
| VIP客户 | 95折 | 累计消费5000元 |
| 至尊客户 | 90折 | 累计消费20000元 |

#### 5.2 客户信息

- 基本信息：姓名、电话、地址
- 交易统计：累计消费、订单数
- 应收账款：欠款金额、信用额度
- 交易历史：所有订单记录

#### 5.3 往来账管理

- 应收账款统计
- 收款登记
- 交易流水查询
- 付款提醒

### 6. 供应商管理

- 供应商基本信息
- 采购记录
- 应付账款
- 付款登记

### 7. 采购入库

#### 7.1 入库流程

1. 选择供应商（可选）
2. 添加商品和数量
3. 输入采购价格
4. 选择付款方式
5. 确认入库

#### 7.2 库存更新

- 入库自动增加库存
- 更新商品成本价
- 记录库存流水

### 8. 统计分析

#### 8.1 销售概览

- 今日/本月销售统计
- 销售趋势图
- 同比环比分析

#### 8.2 利润分析

- 总收入/总利润
- 利润率统计

## 常见问题

### Q1: 数据会丢失吗？

**A**: 所有数据存储在本地SQLite数据库，不会自动丢失。建议定期备份。

### Q2: 换手机怎么迁移数据？

**A**: 
1. 在旧手机上导出数据备份
2. 将备份文件传到新手机
3. 在新手机上恢复数据

### Q3: 支持多用户/多店铺吗？

**A**: 当前版本为单店铺版本，每个设备独立运行，数据不互通。

### Q4: 打印机连接不上怎么办？

**A**: 
1. 确认打印机已开启
2. 确认蓝牙已开启
3. 尝试重新扫描设备
4. 重启打印机和手机

### Q5: 忘记密码怎么办？

**A**: 本版本无需登录，无密码保护。如需数据保护，请使用手机系统锁屏功能。
