### ✨ 主要功能

#### 1. 商品管理
- ✅ 商品信息管理（名称、条码、分类、单位）
- ✅ 多单位价格体系（支持箱→件→盒→个无限层级）
- ✅ 成本价/零售价/批发价多级定价
- ✅ 库存预警设置
- ✅ 商品启用/禁用

#### 2. 库存管理
- ✅ 实时库存查询
- ✅ 入库/出库/调拨操作
- ✅ 库存流水明细（可追溯所有变动）
- ✅ 库存盘点功能
- ✅ 库存成本核算

#### 3. 销售开单
- ✅ 快速开单界面
- ✅ 扫码添加商品（摄像头扫码）
- ✅ 手动改价/折扣
- ✅ 赊账支持
- ✅ 退货退款处理
- ✅ 小票打印预览

#### 4. 客户管理
- ✅ 客户信息管理
- ✅ 客户分级（普通/银卡/金卡/钻石）
- ✅ 不同等级自动应用不同价格
- ✅ 应收账款管理
- ✅ 消费统计

#### 5. 供应商管理
- ✅ 供应商信息管理
- ✅ 采购记录管理
- ✅ 应付账款管理

#### 6. 统计分析
- ✅ 销售趋势图（日/周/月）
- ✅ 利润分析
- ✅ 商品销售排行
- ✅ 客户消费排行
- ✅ 日期范围筛选

#### 7. 系统功能
- ✅ 店铺信息设置
- ✅ 数据备份/恢复
- ✅ 操作日志审计
- ✅ 蓝牙打印机支持（热敏小票）

### 🛠 技术栈

- **前端框架**: Expo 54 + React Native + TypeScript
- **UI 组件**: React Native Elements + 自定义组件
- **数据库**: SQLite (expo-sqlite)
- **状态管理**: React Hooks + Context API
- **路由**: Expo Router
- **图标**: FontAwesome6
- **动画**: React Native Reanimated
